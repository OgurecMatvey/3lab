# Учебный проект: Лабораторная работа №3 (Анализ данных)

## Описание проекта
Данная программа предназначена для анализа статистических данных и построения прогнозов с использованием метода скользящей средней. Проект реализован на языке Python с использованием графического интерфейса PyQt6.

## Команда и варианты
* **Матвей** — Разработка общего интерфейса программы и Вариант 17.
* **Николай** — Вариант 11 (Туризм).


## Функционал
* Загрузка данных из форматов CSV, JSON и XLSX.
* Отображение данных в виде таблицы.
* Расчет исторической скользящей средней и экстраполяция (прогноз) данных.
* Визуализация графиков с помощью Matplotlib.
* Экспорт построенных графиков в форматы PNG и PDF.
